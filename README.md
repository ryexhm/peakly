# PEAKLY - TEAM TAKIS
"Level up together"
An all in one hub centered around building connections through fitness to encourage the youth to live healthier lives.

## HOW TO RUN
Open command prompt in the parent directory and run (pip is required)
```ruby
pip install -r requirements.txt
```
Once installed run the `run.cmd` file to deploy the web app locally to both localhost (127.0.0.0) and your private IP address. (port: 80)

### WARNING
Web app is not optimized for landscape display and may lead to unexpected behaviour.
